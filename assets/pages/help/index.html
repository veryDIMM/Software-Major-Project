QR Code Reader
In order to use the QR code feature of the Rewards System you must have an iPhone or Android device capable of QR reading in the camera app or a third party one from the app store. If you are still having trouble, download the video guide here.
Create App Shortcut
You can create a homescreen shortcut to access the Rewards System website. Note, it is not an app, just a link to the website.
